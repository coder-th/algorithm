### 二进制
a^a = 0
a^0 = a
(a^b)^c = a^(b^c)

a&a = a
a&(a-1): 代表将a最后一位1变为0  0000 1010 => 0000 1000
a&(-n): 代表得到a最后一位1的数字 0000 1010 => 0000 0010
`a&( 1<<n )`: n代表每一位进制所在的位置, 比如: a&(8): 代表查询出第三(2^3)位所在的数字

a|a = a

**a|某个数**,可以设置某些信息在某个进制位。
**a&某个数**，可以查询某个进制位的信息
